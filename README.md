# 🕊️ Манифест нового мира

Одностраничный сайт-манифест 

🔗 **Демо:** [antonderiabinjs.github.io/MANIFEST](https://antonderiabinjs.github.io/MANIFEST)

---

## 🎨 Дизайн и визуальная концепция

| Элемент | Реализация |
|---------|------------|
| **Тёмная тема** | Глубокий тёмно-зелёный фон (`#0a120a`) с золотыми акцентами |
| **Типографика** | Cormorant Garamond (заголовки) + Open Sans (текст) |
| **Параллакс-эффект** | Фиксированный фон с размытием на секциях-цитатах |
| **Анимации** | Плавное появление элементов при скролле, плавающая иконка, bounce-стрелка |
| **Адаптивность** | Полная поддержка мобильных устройств (&lt; 768px и &lt; 480px) |

## 🛠 Технологии

| Технология | Применение |
|------------|------------|
| **HTML5** | Семантическая разметка, якорная навигация |
| **CSS3 (встроенные стили)** | CSS-переменные, Grid, Flexbox, медиа-запросы, keyframes-анимации |
| **Vanilla JavaScript** | Эффект прозрачности навигации при скролле, IntersectionObserver для анимации появления |
| **Google Fonts** | Cormorant Garamond (400, 600, 700, italic) + Open Sans (300, 400, 600, 700) |
| **Open Graph** | Превью для соцсетей (`bak_preview.jpg`) |

---

## ✨ Особенности реализации

- **Полностью статичный** — один HTML-файл, нет зависимостей
- **CSS-переменные** для единой цветовой палитры и типографики
- **Плавный скролл** (`scroll-behavior: smooth`) и якорная навигация
- **Параллакс-секции** с размытым фоном для цитат
- **Адаптивные таблицы** с горизонтальной прокруткой на мобильных
- **Таймлайн** многоэтапного отбора с декоративной линией
- **Карточки с hover-эффектами** — подъём, тень, градиентная линия сверху
- **Мобильное меню** — навигация скрывается на экранах &lt; 768px
